## Features

| Feature                        | Status | Description |
|-------------------------------|--------|-----------|
| Pixel-Perfect UI              | Done   | Exact WhatsApp Web design (2025) |
| Dark / Light Mode Toggle      | Done   | Beautiful dark mode with smooth transition |
| Real Voice Messages           | Done   | Hold mic → record → send (WebM) |
| Typing Indicators             | Done   | Animated "typing..." with dots |
| Online / Last Seen            | Done   | Real-time online status |
| Add New Contacts              | Done   | Add people with name + phone |
| Search Contacts               | Done   | Instant search with highlight |
| Emoji Picker                  | Done   | 100+ emojis with categories |
| Message Timestamps & Checks   | Done   | Single/Double/Blue ticks |
| Responsive (Mobile + Desktop) | Done   | Works perfectly on all devices |
| OTP Login (Phone Auth)        | Done   | Real WhatsApp-style login |
| Smooth Animations             | Done   | Tailwind + Framer Motion feel |
| One File App                  | Done   | Entire app in `App.jsx` (magic!) |
